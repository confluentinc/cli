confluent update
----------------

Description
~~~~~~~~~~~

Update the Confluent CLI.

::

  confluent update [flags]

Flags
~~~~~

::

  -y, --yes         Update without prompting.
      --major       Allow major version updates.
      --no-verify   Skip checksum verification of new binary.

Global Flags
